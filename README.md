# Mock-bit-interview
### How to run?
1. Clone repository
2. Open in VS code.
3. Open terminal and split it.
4. In first terminal:
   ```
   cd client
   npm i
   num run dev
   ```
5. In second termianl:
   ```
   cd server
   pip install -r requirements.txt
   pip install flask_cors
   python app.py
   ```
6. Client end will be running on port: http://localhost:3000/
